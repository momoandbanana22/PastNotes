# テスト計画

## 優先度: 高

1. ✅ **Misskey APIクライアントの初期化テスト**
   - APIクライアントが正しく初期化されること
   - 必要な設定値が設定されること

2. ✅ **API認証テスト（トークン認証）**
   - トークンを使用して認証が成功すること
   - 無効なトークンで認証が失敗すること

3. ✅ **期間指定でノートを取得するテスト**
   - 指定期間内のノートが正しく取得されること
   - 期間外のノートが除外されること

4. ✅ **取得したノートのデータモデルテスト**
   - ノートデータが正しくモデル化されること
   - 必要なフィールドが含まれていること

## 優先度: 中

5. ✅ **ノートをローカルファイルに保存するテスト**
   - ノートが正しい形式でファイルに保存されること
   - 既存ファイルに上書き保存できること

6. ✅ **保存したノートを読み込むテスト**
   - 保存したファイルからノートが正しく読み込めること
   - ファイルがない場合のエラーハンドリング

7. ✅ **ノートをキーワードで検索するテスト**
   - キーワードを含むノートが正しく検索されること
   - 大文字小文字の区別が適切に行われること

8. ✅ **ノートを日付で絞り込むテスト**
   - 指定した日付範囲のノートが正しく絞り込まれること
   - 日付の境界条件が正しく処理されること

9. ✅ **APIエラーハンドリングのテスト**
   - APIエラーが適切に処理されること
   - ネットワークエラー時のリトライ処理

## 優先度: 低

10. ✅ **無効な期間指定のバリデーションテスト**
    - 無効な日付範囲が適切にバリデーションされること
    - エラーメッセージが適切に表示されること
